mac-cleaner-cli is a free and open-source tool designed to help you clean your Mac directly from the terminal. It scans for junk files, caches, logs, and more, making it easy to free up valuable space on your device. This guide will help you download and run mac-cleaner-cli with ease.

## üöÄ Getting Started

### Prerequisites

To use mac-cleaner-cli, you need to have the following:

- A Mac computer with macOS (version 10.15 or later recommended)
- Access to the terminal

### System Requirements

- MacOS version: 10.15 (Catalina) or later
- At least 100 MB of free disk space for the application
- Basic familiarity with using the terminal is helpful, but not required

## üñ•Ô∏è Running mac-cleaner-cli

1. **Open Terminal**  
   You can find Terminal in your Applications folder under Utilities.

2. **Run mac-cleaner-cli**  
   Type `mac-cleaner-cli` in the terminal and press Enter. You may need to provide additional commands based on what you wish to clean.

3. **Use Commands**  
   Here are a few commands to get you started:
   - `mac-cleaner-cli scan`: Scan for junk files.
   - `mac-cleaner-cli clean`: Remove identified junk files.

### Example Usage

To scan for junk files and clean them, you would type:

```bash
mac-cleaner-cli scan
mac-cleaner-cli clean
```
